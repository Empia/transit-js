transit.reader transit.reader( type, opts )
src/com/cognitect/transit.js:49

Create a transit reader instance.

Parameters:
  • type <String | Null>

    type of reader to construct. Default to "json". For verbose mode supply "json-verbose".

  • opts <Object | Null>

    reader options. A JavaScript object to customize the writer Valid entries include "defaultHandler", and "handler". "defaultHandler" should be JavaScript function taking two arguments, the first is the tag, the second the value. "handlers" should be an object of tags to handle. The values are functions that will receive the value of matched tag.


Returns: <transit.reader>

A transit reader.
